# DevTrack

DevTrack é uma plataforma para acompanhar a evolução de desenvolvedores, permitindo registrar tecnologias, cursos, projetos, sessões de estudo e metas em um único lugar.

## Objetivo

Criar uma aplicação completa utilizando o ecossistema .NET para praticar conceitos de desenvolvimento back-end e arquitetura de software.

## Tecnologias

- C#
- ASP.NET Core Web API
- Swagger (OpenAPI)
